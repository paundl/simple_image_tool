# AI generated image tool

## current feature set
* select outtakes and move to OUTTAKES folder
  * select a fodler with images (File -> Open Folder)
  * Navigate images with arrow keys
  * Tag outtake images with key [1]
  * Move outtake images to folder OUTTAKES (Actions -> Move to OUTTAKES)
  * Hash check for moved files before deleting the originals
